About

Built a Java-based inventory management system using object-oriented design. Implemented efficient data structures (including hash maps and trees) to improve lookup speed and organization. Connected the application to a MySQL database to manage inventory, orders, and customer data in a structured way.

What it includes
  • Inventory CRUD (add, update, remove, search)
  • Order tracking and customer records
  • Fast search via data structures
  • Persistent storage via MySQL
